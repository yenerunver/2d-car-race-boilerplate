# 2d-car-race-boilerplate
This is a web-based 2D to-view car racing game boilerplate, built with:
- ViteJS
- ReactJS
- TailwindCSS

## Get Started
First, install the dependencies:
```shell
yarn
```
Then, to run the game on local environment:
```shell
yarn dev
```
